Core: 
    - List Blogs Page (GET /blogs?filters...) [Paginated list of blogs]
    - Get Blog by ID (GET /blogs/{id})  [Blog details]
    - Create Blog (POST /blogs)     [Blog-ID if sucess else error]
    - Update Blog (PUT /blogs/{id}) [request status]
    - Delete Blog (DELETE /blogs/{id})  [requests status]
